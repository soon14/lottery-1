一、相关定义

1、用户：指在本平台注册登记，并得到本平台认可，有资格享受本平台服务的注册会员。

2、委托投注：指用户以注册的用户名单独委托本平台投注购买某彩种，用户对所委托投注成功的彩票拥有完整的奖金获得权利。

3、账户：指用户按本平台的要求，在网站上设立的账户，专门用于存放预付款。

4、预付款：指用户存放在账户中的款项，专门用于委托本平台投注购买用户指定的彩票，以及本平台提供的其他服务和产品。

5、用户注册资料：指用户在注册时提供的个人信息，包括但不限于：用户名、注册账户密码、银行卡账户及密码、用户真实姓名、身份证号码等。

四、追号说明

追号服务指本平台按用户订单，连续多期帮助用户向同一个（组）指定号码提供委托投注的服务。在此服务下，用户可以选择提前预定一个（组）号码进行多期投注，免除时时紧盯电脑之苦，又可以在中奖后自动取消掉后面的投注，节约购彩资金。

1、用户在线发出追号订单，提交自选号码、投注期次、是否基于中奖而终止追号等具体服务要求，并根据本平台提供的服务流程进行足额付款，付款金额为完成该笔追号订单所需的资金总额。

2、本平台将于用户付款后开始执行该订单，完成追号的委托投注服务。

3、用户可选择中奖后或者盈利后系统自动停止追号服务，即如果追号中奖或者盈利，则追号订单中的剩余期次的投注将停止。

4、用户亦可以在订单有效期内通过“我的账户-购彩记录-追号记录”通道随时取消未完成的任一期追号服务或者终止所有未完成的追号服务。

5、在追号服务终止后，用户根据本条第一项支付的款项将根据追号订单的具体执行情况进行结算，系统会将未执行追号期次的购彩金额退至用户账户中。

6、用户使用追号服务的同时，不影响其使用本平台其他服务。

五、兑奖流程

每期彩票开奖完成后，本平台将公布中奖号码，并把税后奖金在开奖后7个工作日内直接返入中奖者在本平台的账户中。您使用本平台服务即视为您同意代为保管彩票、兑奖、领取奖金。（一个工作日内能否完成兑奖返款，流程是否为用户授权本平台兑奖后再将税后奖金直接返入中奖者在平台的账户，因为本平台无权直接销售彩票，仅为受托投注购买服务，这段话可以更改为：“每期彩票开奖完成后，本平台将公布中奖号码，最终开奖结果将以官方彩票发行机构和销售机构公布的中奖情况为准。在彩票中奖情况下，由本平台负责代理兑奖、领取奖金事宜，并在开奖后的7个工作日内把税后奖金直接返入中奖用户在本平台的账户。您使用本平台服务即视为您同意代为保管彩票、兑奖、领取奖金。”
