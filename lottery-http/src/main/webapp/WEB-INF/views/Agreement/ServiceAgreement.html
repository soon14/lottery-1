三、用户帐号冻结、暂停和终止使用

1.用户帐号冻结

帐号冻结指用户可以正常登录本平台，但无法对其帐号进行任何操作，用户同意出现如下情况本平台有权不经通知单方面判断并对其帐号进行冻结：

（1）本平台其他服务说明及约定中所涉及的帐号冻结条件。

2.用户帐号暂停

帐号暂停指用户暂时无法正常登录本平台，暂时无法对其帐号进行任何操作，用户同意出现如下情况本平台有权不经通知单方面暂停用户使用其帐号权利：

（3）本平台系统公告出暂停服务的情况。

（4）本平台其他服务说明及约定中所涉及的帐号暂停条件。

3.用户帐号终止使用

帐号终止使用指用户的帐号已被本平台收回并停止服务，用户的行为过程中没有给本平台造成损失，如果用户的行为已经给本平台造成损失，用户需要承担相应经济责任，用户同意出现如下情况本平台有权先采，用帐号冻结处理方式要求其对如下行为进行解释，如5个工作日内用户没有合理解释本平台有权终止其帐号：

（3）未经本平台许可，用户擅自利用本平台所提供的服务进行经营。
